Welcome WEB DEVELOPER !

To an introductory guide of THREE.JS

My name is Rahul Ravi. I am earning my bachelors degree in Cognitive and Data Science as an undergraduate student at UC San Diego.

The purpose of this website is to uncover the tools provided by three.js. As a new web developer you have heard of HTML, CSS, and JavaScript. These are the monoliths of web development and requirements for most front-end web developer jobs. Three.js is an open source JavaScript library licensed by MIT and it can be used to create three dimensional environments in website for users to interact with. Remember back when you needed Abobe Flash to play games in a browser? Now the technology that allows for this can be achieved through javascript as well. There are multiple distinguished websites that utilize three.js to offer a better user experience.

The demand for interactive, modern, and functional web interfaces is large, growing and as technology for virtual and augmented reality becomes more socially orthodox, developers will benefit from utilization of advanced libraries like three.js. Do you want to explore cool projects and learn more about three.js so you can use it to build a fresh interface for your portfolio, business, or job?
